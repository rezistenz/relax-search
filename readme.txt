Проект веб-сервиса для поиска списка самых популярных фирм, 
отсортированных по рейтингу фламп.

Веб-сервис построен на основе технологии jax-rs 2.0, 
в качестве реализации используется библиотека jersey версии 2.
Для сериализации java объектов в json используется библиотека jackson.
  

Для сборки проекта необходимо:
	- JDK 1.7
	- maven2

Сборка проекта:
	mvn clean package
	
Запуск проекта:
	mvn tomcat:run
	
Либо сделать деплой собранного пакета relax-search.war в экземпляр сервера tomcat.
	
Пример обращения к web сервису:
	curl http://localhost:8080/relax-search/search?what=кинотеатр

Конфигурация находится в файле src/main/resources/config.properties
Свойства для конфигурации:
	org.rezistenz.relax.search.RelaxSearchService.wheres - список мест (городов) где осуществлять поиск, через разделитель ',' - запятая.
	org.rezistenz.relax.search.RelaxSearchService.key - ключ доступа к сервисам поиска.
